 🛠️ EliteSys Manager

Windows sistemler için geliştirilmiş, otomatik kontrol ve yönetim işlemleri sunan gelişmiş bir komut dosyasıdır.

 ⚙️ Özellikler

* ✔️ Otomatik **Yönetici Yetkisi (Admin) kontrolü**
* ✔️ Sadece **Windows 10 / 11 desteği**
* ✔️ **64-bit (x64)** sistem kontrolü
* ✔️ Hatalarda kullanıcı dostu uyarılar
* ✔️ Tek dosya ile hızlı kullanım

---

 💻 Sistem Gereksinimleri

* Windows 10 veya Windows 11
* 64-bit (x64) mimari
* Yönetici yetkisi (script otomatik ister)


 📥 Kurulum / Kullanım

 1. Dosyayı indir

Aşağıdaki komutu çalıştırarak script'i masaüstüne indirebilirsiniz:

```bat
curl -L -o %USERPROFILE%\Desktop\EliteSys_Manager.bat https://raw.githubusercontent.com/yakup-alan/EliteSys/main/EliteSys_Manager.bat
```

---

 2. Çalıştır

* Masaüstüne inen dosyaya çift tıklayın
* Script otomatik olarak:

  * Yönetici yetkisi ister
  * Sistem uyumluluğunu kontrol eder
  * Ardından çalışmaya başlar

---

 ⚠️ Önemli Notlar

* Script yalnızca **Windows 10/11 x64** sistemlerde çalışır
* Yönetici yetkisi olmadan çalışmaz (otomatik yükseltir)
* Güvenlik açısından script içeriğini incelemeniz önerilir

---
## 📌 Sorumluluk Reddi

Bu araç eğitim ve sistem yönetimi amaçlıdır. Kullanım sonucu oluşabilecek sorunlardan kullanıcı sorumludur.

```bat
curl -L https://raw.githubusercontent.com/yakup-alan/EliteSys/main/EliteSys_Manager.bat -o %USERPROFILE%\Desktop\EliteSys_Manager.bat && start %USERPROFILE%\Desktop\EliteSys_Manager.bat
```
